# Arab Football Fans Social Platform ⚽

A social platform designed for Arab football fans, aiming to bring football lovers together in one place to follow matches, share predictions, interact with live events, publish posts, and discuss teams, players, and tournaments.

## Project Idea

This application is a sports-focused social media platform centered around football. Users can follow match updates in real time, interact with posts, write comments, follow other users, and communicate through direct chats.

The platform also includes a match prediction system, adding an interactive and competitive experience for users.

## Main Features

- User registration and login.
- Create and share sports-related posts.
- Comment on posts and interact with them.
- Follow other users.
- Chat system between users.
- Follow live match events.
- Predict match results.
- View match and team information.
- A customized experience for Arab football fans.

## Project Goal

The goal of this project is to build an Arabic sports community that brings football fans together in one platform, allowing them to communicate, discuss matches, share predictions, and follow sports events in an easy and interactive way.

## Target Audience

- Arab football fans.
- Users interested in following matches and tournaments.
- Users who like sharing sports opinions and analysis.
- Fans who enjoy match predictions and sports challenges.

## Application Sections

### Posts

Users can publish their opinions and analysis about matches, teams, players, and tournaments.

### Comments and Interactions

The application allows users to comment on posts and interact with them, helping create sports discussions between users.

### Following

Users can follow other accounts to keep up with their posts and updates.

### Chats

The application provides a chat system that allows users to communicate directly with each other.

### Match Predictions

Users can predict match results before matches start, adding a fun and competitive experience between fans.

### Live Match Events

The application displays live match events such as goals, cards, substitutions, and other important match details.

## Technologies Used

- **Frontend:** Flutter
- **Backend:** ASP.NET Core Web API using C#
- **Database:** SQL Database
- **API:** RESTful API
